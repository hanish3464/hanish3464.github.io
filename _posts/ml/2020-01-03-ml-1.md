Week 0


Supervised learning – 지도식 학습 방법 = 선생님이 학생을 교육하듯 답을 아는 상태에서 학습.


Ex) Housing Price Prediction



<img src="https://user-images.githubusercontent.com/51018265/71723815-5b875780-2e71-11ea-9a10-da5767fcc203.png" width="60%">
  

집의 사이즈에 따른 가격을 분석 => 평수에 따른 가격이 어떻게 구성 되는지를 알고 있음.


Machine이 집의 사이즈에 대한 data를 학습하고 결과 값으로 집의 매매값이 나타남.


그리고 이를 실제 값과 비교하여 Cost(오차)를 알 수 있음.


집의 사이즈가 커질수록 매매가가 높아지기 때문에 “연속적인 값”으로 나타남.


이러한 연속적인 값으로 나오는 것을 Regression이라고 함.


반대로 집의 구조에 따라 아파트인지 빌라인지 단독주택인지 구분하는 problem을 Classification이라고 함.


Ex) Breast Cancer 종양 사이즈에 따른 악성종양 여부 구분

이렇듯 data들을 구분한다는 것은 어떤 기준이 필요하다는 것이고 위와 같은 linear한 기준에 의해 classification 될 수 있음.


또한 이러한 잘 구분된 선을 찾는 것이 supervised machine 이 학습을 통해 찾아내는 결과가 됨.



Regression vs Classification


두가지의 학습 방식은 다른 성향의 결과 값을 예측하고자 할 때 사용이 됨. 따라서 무엇을 예측하고자 하는지를 적절히 파악하여 사용해야 함.


Unspervised Learning = 자율학습이라고 표현되는 학습 방법임. 답을 모르는 상태에서 하습을 하는 경우에 해당함.


Dataset에 어떤 정보가 있는지 알지 못하고 어떤 data로 구성이 되어 있는지도 모를 때 사용.


사용자가 원하는 결과를 비슷하고 semantic한 정보로 그룹핑하여 분류하여 보고자 할 때 사용함.


이를 Clustering 이라고도 함.


Ex) 비슷한 뉴스나 피드들을 grouping 하는 서비스 or 음성인식에서의 Cocktail party problem algorithm


Cocktail party problem algorithm – 칵테일 파티와 같이 시끄러운 곳에서 듣고자 하는 원하는 음성만을 분리해내고


잡음들을 무시함으로써 깨끗한 음성만을 인식할 수 있게 해줌.
